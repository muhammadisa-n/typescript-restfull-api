# typescript-restfull-api

# Setup Project

Create .env file

```
DATABASE_URL="mysql://username:password@host:port/databasename"
```

```shell
npm install
```

```shell
npx prisma migrate dev
```

```shell
npx prisma generate
```

# Run Development

```shell
npm run dev
```

# Build To Production

```shell
npm run build
```

```shell
npm start
```
